Hybrid handheld application for anatomical visualization with two interchangeable modes — AR Mode and 3D non-AR Mode — switchable via a dynamic toggle button. 
Supports touch-based rotation, translation, and scaling in 3D mode, along with the ability to show/hide individual anatomical structures. 
UI buttons dynamically reflect structure status (active/inactive) with linked updates to labels, materials, and slice views, wrapped in expandable panels with smooth animations. 
Designed as a portable education tool — runs on any mobile device or tablet with a camera — for explaining anatomy or surgical concepts to patients or students, with potential to evolve into a preoperative planning tool for visualizing bone defects, implant sites, and surgical pathways.
